# Perpustakaan Puteri Seroja - Live Admin Dashboard

An open-source administration workspace system connecting an adaptive frontend web layout UI seamlessly to MySQL backends managed under phpMyAdmin.

## Local Installation Setup

1. Clone this repository or download the source code zip bundle directly to your workspace computer.
2. Install **XAMPP** (or any environment tool containing Apache and MariaDB/MySQL servers).
3. Move the downloaded files directly inside your local root hosting directory at `C:/xampp/htdocs/perpustakaan/`.
4. Launch your browser engine dashboard and navigate to your interface tool at `http://localhost/phpmyadmin/`.
5. Create a new clean schema workspace database structure named `perpustakaanpuetriseroja`.
6. Use the **Import** mechanism option on top of the tab toolbar list inside phpMyAdmin to select and upload your original text dump schema `perpustakaanpuetriseroja 2.sql`.
7. Once successfully uploaded, access your web dashboard system deployment interface locally inside your web browser via: `http://localhost/perpustakaan/index.html`.
